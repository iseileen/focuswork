# FocusWork
Version 0.1.0

Capture fast. Find fast. Follow up fast.

FocusWork 是一個以個人工作管理為目的的輕量工具，專注於快速記錄、快速查詢與追蹤待辦事項。

## 功能

- Today
- Waiting
- Inbox
- This Week
- Quick Add
- Search
- Daily Review
- JSON 匯出 / 匯入

## 技術

- HTML
- CSS
- Vanilla JavaScript
- LocalStorage

## 資料儲存

FocusWork 採用 Local-first 設計。

- 資料僅儲存在目前瀏覽器
- 不需要登入
- 不需要後端
- 不支援跨裝置同步
- 建議定期匯出 JSON 備份

## 使用方式

直接開啟 `index.html` 即可使用。

也可以部署至 GitHub Pages 或其他靜態網站服務。
